Stardetect

A simple python script that attempts to create usable extrapolated data from Hubble telescope imagery, by means of image processing.

My motivation for this was to help delve into other areas such as Python and SciPy, and doing so in an area I am very interested in, Astronomy.



Some noteworthy modules used include the following;

  • Python PIL API
  • Matplotlib
