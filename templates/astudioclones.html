Android Studio Journey

Taking it upon myself to learn Android Studio, I learnt to make clones of a few popular tools out there.

The projects all mimicked the core functionality of each tool in its specific unique way.

In all instances where a backend server with database was required, I leveraged the use of ParseDB on a machine hosted through Amazon Web Services.


  • Instagram Clone
  • Whatsapp Clone
  • Twitter Clone
  • Uber Clone
  • Flappy Bird Clone made with LibGDX

Looking at existing tools and mimicking them helps develop skills immensely, and it is a task that I will continue to perform.
