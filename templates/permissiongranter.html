Android Permission Granter

A simple android plugin module that allows Unity to request permissions at runtime.

I developed this as a solution to a problem that was recurring on most Unity projects I encountered, the fact that native android functions as basic as requesting permissions at runtime was not possible from the C#-Unity side.

Any solutions like this on the Unity store exist bundled in unnecessary packages full of unwanted plugins that bloat the project you work in, as well costing a large sum. So I wanted to make an extremely lightweight plugin that can live in each project I work in that requires this functionality.

The project contains the Android Studio native code in Java as well as the Unity and C# source files, documented and explained in examples, to help get you set up. If the plugin is all you are after, the unitypackage is included as well.

Main tools used include the following;

  • Unity3D
  • Android Studio
  • Gradle
