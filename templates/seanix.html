Seanix

This site was developed as a means for me to log and supplement my skills and learning in an externally visible manner.

Alongside the benefits of having an online portfolio, I am able to learn more into the Web Development space and tackle various tools used within the process, thus helping me in two different ways.

The backend is written in Python and makes use of Flask and uWSGI, hosted off an Ubuntu VPS from Digital Ocean.

The frontend is written in basic HTML5, CSS and Javascript, and piggybacks off Bootstrap and Sass.

I initially wrote the backend in ExpressJS with NodeJS, and using PUG, but shortly after rewrote it so as to split the skills between Python and Javascript, as well as the site being extremely minimalistic in terms of the tools required.

I will continue to use this place as a means to delve deeper into the world of Web Development, in the full stack manner.

Main frameworks and libraries used include the following;

  • Bootstrap
  • Flask
  • uWSGI
  • Digital Ocean
