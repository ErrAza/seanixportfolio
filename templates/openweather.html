Openweather

Openweather is a simple Android weather application developed in Android Studio for the purposes of exploring and learning the techniques involved.

It uses the OpenWeatherMap.org API to view weather forecasts at different locations in a rolling 5-day window.

Some noteworthy tech used include the following;

  • OpenWeatherMap API
  • Android Studio
