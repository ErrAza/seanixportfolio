Session

Session is an Android application developed in Android Studio that I am working on when I find the time to do so.

The main idea behind it, is that it will allow users to discover each other, and be paired based entirely on musical interests.

The authorization procedure involves quick login by aid of the ParseDB on AWS. The user can then set up their profile, as well as explore a seemingly infinite directory of artists, courtesy of the LastFM API.

Identifying types of users enables the idea that a user can take on the role of one seeking fellow musicians, so that the aim is not to be a space for matching people, but for a space to aid the formation and networking of local musicians. In doing so, the user can not only find friends with similar musical genre interests, but possibly be involved in the musical industry and help to cultivate it.

The motivation behind the idea is one that resonates with me personally as I noticed there a severe lack of communication and networking in the local music scene, so if you are a musician of some type, it is extremely difficult to find likeminded people and assemble them to just in general, make more local music, with the added benefit of being able to find new artists/bands, and make a few friends along the way.


Some noteworthy tech used include the following;

  • Volley
  • Android Studio
  • ParseDB
  • Amazon Web Services
  • LastFM API
